## Accessibility is a Requirement

Over 1 billion people worldwide have disabilities. Accessible websites improve UX for everyone.

## Tool Comparison

- **axe-core**: Industry standard, zero false positives, 80+ rules
- **Lighthouse**: Built into Chrome, combined performance + a11y audits
- **Pa11y**: CLI-first, excellent for CI/CD pipelines
- **WAVE**: Browser extension for quick audits

## Common Issues to Fix

1. Missing alt text on images
2. Poor color contrast (need 4.5:1 ratio)
3. Missing form labels
4. Keyboard navigation issues
5. Missing focus indicators
