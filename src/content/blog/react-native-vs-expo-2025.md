## React Native: CLI vs Expo

Choose the right approach for your mobile app.

## Expo Managed

- Faster development
- OTA updates
- No native code needed
- Limited native modules

## React Native CLI

- Full native access
- Custom native modules
- More complex setup
- More flexibility

## Recommendation

Start with Expo, eject if needed.
