## Progressive Web Apps

PWAs combine the best of web and native apps.

## Key Features

1. **Installable**: Add to home screen
2. **Offline**: Service worker caching
3. **Push Notifications**: Re-engage users
4. **Fast**: App shell architecture

## Implementation

- Web App Manifest for installation
- Service Worker for offline/caching
- HTTPS required
